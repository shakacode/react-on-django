# Upgrading to Pro

The package is source-available and free for non-commercial or no-revenue use.
Commercial production use requires a paid license from ShakaCode.

When the broader Pro helper surface lands, upgrading will focus on enabling the
features you need rather than switching packages.

Expected steps:

1. purchase a commercial license when required
2. enable the desired Pro-oriented settings
3. adopt the streamed and cached helper APIs incrementally
4. validate the RSC and streaming flows in the example-style test harness

This page will grow alongside the port as more upstream Pro features land.
